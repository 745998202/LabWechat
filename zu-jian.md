# 组件

一个小程序页面由多个部分组成，组件就是小程序的基本组成单元。

组件是在WXML模板文件声明中使用的，WXML的语法和HTML语法相似，小程序使用标签名来引用一个组件，通常包含**开始标签**和**结束标签**，该标签的属性用来描述该组件

```text
<!-- page.xml -->

<image mode = "scaleToFile" src="img.png"></image>
```

{% hint style="info" %}
所有组件都是小写，多个英文单词之间以英文横杠进行连接“-”
{% endhint %}
